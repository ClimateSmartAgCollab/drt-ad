# DRT Agreements Landing Page

Marketing page for the Data Request Tracker (DRT) semantic agreement tooling. Built with React, TypeScript, and Vite.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build   # outputs to dist/
npm run preview # optional local preview
```

## Automation

- GitHub Actions workflow `.github/workflows/deploy.yml` builds `dist/` on every push to `main` and publishes via GitHub Pages.
